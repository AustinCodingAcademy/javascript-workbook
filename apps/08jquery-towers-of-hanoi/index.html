Instructions

Objective

There are three "stacks". The first stack has 4 blocks. The object of the game is to move all four blocks to another stack, in the same order, in as few moves as possible.

Rules

  1. Only the last block in a stack can be moved.
  2. A larger block cannot be stacked on a smaller block.

How To

  • Click a stack to move a block from.
  • Click a second stack to move the block to.
  • Re-click the starting stack to cancel a move.
  • Click 'Reset Game' to start over, or play a new game.
